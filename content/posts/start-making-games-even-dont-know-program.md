*“I want to get started with making games, but I don’t know how. Where should I start?”*

It can be overwhelming when you are starting off, because there are so many options available to you. I mean just look at plethora of game making tools at  [Pixel Prospetor](http://www.pixelprospector.com/the-big-list-of-game-making-tools/), and you’ll see what I’m talking about.  If you have zero programming experience, then I’d suggest starting with [Construct2.](http://www.pixelprospector.com/the-big-list-of-game-making-tools/#construct)

Construct 2 allows you to create HTML5 games without ever having to write code. Everything is handled in the IDE (Integrated Development Environment) which has a GUI (Graphical User Interface) that anyone could use. You simply drag-and-drop objects into your scene, and can make changes to the objects properties by right clicking.  
Even better, it’s free! There are premium versions available, but if you’re just starting off, then the free version offers more than enough to get you going.

The best part of all? You can export to a number of devices, including Windows Phone, Windows 8, and the browser. It’s all built in and part of Construct.
