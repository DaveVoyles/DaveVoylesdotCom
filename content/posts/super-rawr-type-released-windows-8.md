## [Play *Super Rawr-Type* for **FREE** in the Windows 8 Store](http://windows.microsoft.com/en-us/windows/search#q=super+rawr-type)

Super Rawr-Type, my 2D sidescrolling JavaScript shooter, is finally available for Windows 8. I actually finished working on it in August, but needed to change a few lines of code to have it pass certification, and completely forgot to resubmit. On Friday I made the final necessary changes, submitted, and it’s up in the store today.

The project is open source, and [you can find my GitHub here.](https://github.com/DaveVoyles/Super-Rawr-Type-Windows8)

I’ve also made a number of tutorials for the game, [which can be found here.](http://davevoyles.azurewebsites.net/html5-javascript/)
